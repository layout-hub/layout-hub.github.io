About

Here you can browse free layouts for QGIS. All layouts are downloadable, and they should be placed in your layout templates folder for your QGIS user profile. If you want you can submit your own layouts to this repository and they will be included, pending time and a quick review.

This is a static DEMO site and not a permanent solution. That should include a "submit" form for easy contributing, search and filters, showing dynamic content. This requires an active server with PHP, database, etc.

QGIS Layout-Hub

Layout Hub

Collections of free to use QGIS layout qpt-files to download and use. Place downloaded qpt-files in your QGIS user profile 'composer_templates' folder. You can then use them from your 'Layout Manager' in QGIS.
